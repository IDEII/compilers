## Абстрактный синтаксис

Программа состоит из набора функций, где у каждой из них есть заголовок и тело. В каждом заголовке есть
набор параметров или их отсутствие, а так же тип возвращаемого значения или так же его отсутствие.
Все параметры разделены друг с другом запятой.

```
Program -> Function* 
Function -> FunctionHeader "=" FunctionBody 
FunctionHeader -> (NType | "void") VARNAME ("<-" FunctionParams)?
FunctionParams-> NFuncParams ',' NFuncParam | NFuncParam .
```

Тело функции состоит из предложений и заканчивается на точку. Каждое предложение разделяется друг с
другом точкой с запятой.

```
FunctionBody -> Statements "."
Statements -> NStatements ';' NStatement | NStatement 
```

Тело функций состоит из набора предложений, каждое из которых которых может быть либо  каким-либо
оператором, либо циклом.

```
NStatement -> NDeclarationStatement 
| NAssignmentStatement 
| NCallExpression 
| NIfStatement 
| NIfElseStatement 
| NLoopWhileStatement 
| NLoopForStatement 
| NReturnStatement 
| NLoopPostStatement
```

В языке есть 3 базовых тип, а так же их массивы.

```
NType -> NBasicType | NType "[]" .
NBasicType -> KW_INT | KW_CHAR | KW_BOOL .
```

 Переменные можно задать с помощью обозначения типа и имени переменной, а так же во время объявления
можно задать ей значение.

```
 NVarDeclarations -> NVarDeclarations ',' NVarDeclaration 
| NVarDeclaration
NVarDeclaration ->  VARNAME ':=' NArithmeticExpression
|  VARNAME
```

В языке есть все основные операции, и все основные типы констант.

## Лексическая структура и конкретный синтаксис

### Конкретный синтаксис

```
NProgram -> NFunction 
| NProgram NFunction

NFunction -> NFunctionHeader "=" NFuncBody

NFunctionHeader -> NType VARNAME  "<-" NFuncParamStatements
NFunctionHeader -> "void" VARNAME  "<-" NFuncParamStatements 
NFunctionHeader -> "void" VARNAME

NType -> NBasicType | NType "[]" 
NBasicType -> KW_INT | KW_CHAR | KW_BOOL 

NFuncBody -> NStatements '.' 
NFuncParamStatements -> NFuncParams 
NFuncParams -> NFuncParams ',' NFuncParam 
| NFuncParam 

NFuncParam -> NType VARNAME 

NStatements -> NStatements ';' NStatement 
| NStatement 

NStatement -> NDeclarationStatement 
| NAssignmentStatement 
| NCallExpression 
| NIfStatement 
| NIfElseStatement 
| NLoopWhileStatement 
| NLoopForStatement 
| NReturnStatement 
| NLoopPostStatement

NDeclarationStatement -> NType NVarDeclarations

NVarDeclarations -> NVarDeclarations ',' NVarDeclaration 
| NVarDeclaration 
NVarDeclaration ->  VARNAME ':=' NArithmeticExpression
|  VARNAME 

NAssignmentStatement -> NSpec ':=' NExpr 

NSpec -> '(' NType NPrim ')'
| NSpec NPrim 
| NPrim 

NPrim -> NConst |  VARNAME  | '(' NExpr ')' 

NConst -> INTEGER | CHAR | REF_CONST | STRING | KW_TRUE | KW_FALSE 

NExpr ->  NLogicalExpression 
| NExpr '|' NLogicalEXpression 
| NExpr '@' NLogicalEXpression 

NLogicalExpression -> NCompareExpression 
| NLogicalExpression '&' NCompareExpression 

NCompareOperator -> '==' | '!=' | '<' | '>' | '<=' | '=>' 

NCompareExpression -> NCallExpression
| NCompareExpression NCompareOperator NCallExpression 

NCallExpression -> NArithmeticExpression 
|  VARNAME  '<-' NArgs 
NArgs -> NArithmeticExpression | NArgs ',' NArithmeticExpression 

NAdditiveOperator -> '+' | '-' 

NArithmeticExpression -> NMultiplicativeExperssion 
| NArithmeticExpression NAdditiveOperator NMultiplicativeExperssion 


NMultiplicativeExperssion -> NTerm 
| NMultiplicativeExperssion NMultiplicativeOperator NTerm 

NMultiplicativeOperator -> '*' | '/' | '%' 

NTerm -> NFactor | NFactor '^' NTerm 

NFactor -> '!' NSpec | '-' NSpec | NSpec 

NIfElseStatement -> NExpr 'then' NStatements NElseBlock '.' 

NIfStatement -> NExpr 'then' NStatements '.' 

NElseBlock -> 'else' NStatements 

NLoopWhileStatement -> NExpr 'loop' NStatements '.' 
NLoopForStatement -> NCycle NStatements '.' 
NLoopPostStatement -> 'loop' NStatements 'while' NExpr '.' 

NCycle -> NExpr '~' NExpr 'loop' NCycleVar 
NCycleVar ->  VARNAME 

NReturnStatement -> 'return' NExpr 
```

### Лексическая структура

```
INTEGER = [A-Za-z0-9]+\\{[0-9]+\\}\\$|[0-9]+
CHAR = \\$\\"[A-Za-z0-9]\\" 
STRING = \\".*\\"  
VARNAME = \\{[A-Za-z0-9]*\\}  
REF_CONST = null'
```

NVarDeclarations |= NVarDeclaration, lambda decl: [decl]

# комментарии вида ##

parser.add_skipped_domain('\\#\\#.*')
